## Features

- Isometric room rendering with dynamic depth sorting
- Multi-level stair system with automatic detection
- Avatar movement with pathfinding
- Habbo-compatible asset loading
- Wall rendering with door occlusion

## Quick Start

```bash
npm install
npm run dev
```

## Tech Stack

- Phaser 3
- TypeScript
- Vite
